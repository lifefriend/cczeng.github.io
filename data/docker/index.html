publish spring-boot-web use docker

1、package a spring-boot web

2、copy jar to centos

3、docker build

a. mkdir FOLDER-NAME

b. touch dockerfile

  
                    # vi dockerfile

                    # base image
                    FROM openjdk:8
                    # author
                    MAINTAINER YOUR-NAME
                    # app info
                    LABEL app="APP-NAME" version="0.0.1" by="YOUR-NAME"
                    # build app name
                    COPY ./demo-0.0.1-SNAPSHOT.jar DOCKER-APP-NAME.jar
                    # run app
                    CMD java -jar DOCKER-APP-NAME.jar

c.build app


                    docker build -t BUILD-NAME .

d.check image

docker images

4、run image


                    docker run --name alias-name -p 8080:8080 -d build-name (or image-id)

5、check logs


                    docker logs alias-name (or container-id)
